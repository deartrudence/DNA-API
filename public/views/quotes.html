Information Available in the API

EndPoints

All Quotes

http://www.dontpanic.space/api/v1/quotes

Returns all the quotes available in the API

Random Quote

http://www.dontpanic.space/api/v1/quotes/random

Returns one random quote

Single Quote by Id

http://www.dontpanic.space/api/v1/quotes/id/55a5af5643c98f11004daa14

Returns a single quote by ID

All Tags

http://www.dontpanic.space/api/v1/quotes/tags

Returns a list of all tags that have been used to tag a quote in the API

All Books

http://www.dontpanic.space/api/v1/quotes/books

Returns a list of all books where quotes have been taken from

Quotes by Tag(s)

http://www.dontpanic.space/api/v1/quotes/tag/universe

Or

http://www.dontpanic.space/api/v1/quotes/tag/impossible&improbable

returns all quotes tagged by a specific tag or tags when separated by '&'

Quotes by Book

http://www.dontpanic.space/api/v1/quotes/book/The%20Hitchhiker's%20Guide%20to%20the%20Galaxy

returns all quotes from the specified book
